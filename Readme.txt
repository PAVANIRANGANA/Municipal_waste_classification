# CleanTech: Transforming Waste Management with Transfer Learning

## Overview
A Flask web app that classifies waste images into Biodegradable or Non-Biodegradable using VGG16.

## Setup Instructions

1. Install dependencies:
   pip install flask tensorflow pillow

2. Ensure the model file `vgg16.h5` is in the root directory.

3. Run the app:
   python app.py

4. Visit `http://127.0.0.1:5000` in your browser.

## Folder Structure
- static/uploads: where uploaded images are stored
- templates/: HTML templates
